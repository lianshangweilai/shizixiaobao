儿童识字小报生成器

基于 AI 的儿童中文识字学习海报生成工具

界面预览版
主要功能
  • 基于 Kie AI Nano Banana Pro 的图像生成
  • 内置 6 大主题场景词汇库（超市、医院、公园等）
  • 自动生成 15-20 个带拼音标注的中文词汇
  • A4 竖版格式，适合打印和学习
  • Web 界面操作，简单易用
  • 实时预览提示词，可自定义参数
  • 生成历史记录，方便查看和管理
使用提示
  1. 需要配置 Kie AI API Key
  2. 生成过程约需 30-60 秒
  3. 支持多种图片格式和分辨率
  4. 适合 5-9 岁儿童使用

  1. 双击 一键测试.bat 运行测试版

  3. 在浏览器中访问 http://localhost:5000
  4. 选择场景和标题，点击生成
