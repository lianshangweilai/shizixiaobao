# 儿童识字小报生成器 - 快速开始指南

## 🚀 快速启动

### 方法1：双击运行（推荐）
1. 双击 `run.bat` 文件启动程序
2. 按照提示进行操作

### 方法2：命令行运行
```bash
# 查看帮助
python src/main.py --help

# 交互式生成（推荐新手）
python src/main.py -i

# 直接指定主题和标题
python src/main.py -t 超市 -T "走进超市"
```

## 📋 使用前准备

### 1. 获取API Key
- 访问：https://kie.ai/api-key
- 注册并获取您的API Key

### 2. 配置API Key
- 打开 `.env` 文件（如果没有会自动创建）
- 将 `your_api_key_here` 替换为您的实际API Key：
  ```
  KIE_AI_API_KEY=您的API密钥
  ```

## 🎯 支持的场景

当前内置的场景包括：
- 🏪 **超市** - 收银员、货架、推车、商品等
- 🏥 **医院** - 医生、护士、药品、病床等
- 🌳 **公园** - 花草、树木、秋千、滑梯等

- 🦁 **动物园** - 各种动物、饲养员、笼子等
- 🚂 **火车站** - 列车、站台、乘客、行李等

## 💡 使用示例

### 交互式生成
```bash
python src/main.py -i
```
程序会依次询问：
1. 主题/场景（如：超市）
2. 小报标题（如：走进超市）

### 批量生成不同场景
```bash
# 生成超市主题
python src/main.py -t 超市 -T "走进超市" -o ./outputs/超市/

# 生成医院主题
python src/main.py -t 医院 -T "快乐医院" --resolution 2K

# 生成公园主题
python src/main.py -t 公园 -T "美丽的公园" --ratio 3:4 --resolution 2K
```

## 📁 输出文件
- 生成的图片默认保存在 `outputs/` 目录
- 生成历史记录在 `outputs/history/` 目录
- 文件名格式：`generated_时间戳.png`

## ⚠️ 注意事项
1. 图片生成需要30-60秒时间
2. 需要稳定的网络连接
3. 请确保API Key有效且有足够余额

## ❓ 常见问题

**Q: 提示"未找到API Key"**
A: 检查 `.env` 文件是否存在并正确填写了API Key

**Q: 生成失败怎么办？**
A: 检查网络连接，确认API Key有效，查看错误信息

**Q: 可以添加新场景吗？**
A: 可以！在 `config/vocabulary_data/` 目录创建JSON文件添加新场景词汇
