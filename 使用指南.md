## 📦 项目文件说明

### 1. 演示版本
- `在线演示版.html` - 纯前端演示，无需API Key

### 2. 正式版本
- `正式版.html` - 完整界面（需要本地服务器）
- `启动正式版.bat` - 一键启动Web服务器
- `web/full_version.py` - 完整的Flask Web服务器

### 3. 辅助工具
- `快速测试.py` - Python环境检查
- `检查状态.bat` - 文件完整性检查

## 🚀 快速开始

### 方法1：使用正式版（推荐）

1. **获取API Key**
   - 访问 https://kie.ai/api-key
   - 注册账号并获取API Key

2. **配置API Key**
   - 创建 `.env` 文件（或使用已有模板）
   - 添加内容：`KIE_AI_API_KEY=您的API密钥`

3. **启动服务**
   ```
   双击：启动正式版.bat
   ```

4. **开始使用**
   - 浏览器自动打开 http://localhost:5000
   - 配置API Key（如果还未配置）
   - 选择场景、输入标题
   - 点击生成

### 方法2：纯前端演示

```
双击：在线演示版.html
```

- 无需安装和配置
- 可以预览界面效果
- 无法生成真实图片

## 🎯 功能特点

### ✅ 已实现功能
1. **场景选择** - 6大主题场景
2. **词汇生成** - 自动生成15-20个词汇
3. **提示词预览** - 查看AI生成指令
4. **参数调整** - 分辨率、宽高比、格式
5. **实时生成** - 30-60秒生成时间
6. **结果下载** - 高清图片下载
7. **历史记录** - 保存生成历史

### 🔄 工作流程
1. 选择场景（超市、医院等）
2. 输入小报标题
3. 调整生成参数
4. 预览提示词（可选）
5. 点击生成
6. 等待生成完成
7. 下载高清图片

## 💡 使用技巧

### 1. 标题建议
- 使用6-8个字的标题
- 如：《走进超市》、《快乐医院》

### 2. 参数选择
- **宽高比**：3:4 适合A4打印
- **分辨率**：2K 平衡质量和速度
- **格式**：PNG无损质量最佳

### 3. 词汇说明
每个场景包含：
- **人物** - 场景中的角色
- **物品** - 常见物品
- **设施** - 相关设施
- **环境** - 环境元素

## ❓ 常见问题

### Q: 提示"未找到Python"
A: 需要安装Python 3.8+，访问 https://www.python.org/downloads/

### Q: 提示"依赖包未安装"
A: 运行 `pip install flask python-dotenv requests tqdm colorama`

### Q: 生成失败
1. 检查API Key是否正确
2. 确认网络连接正常
3. 检查API配额是否充足

### Q: 端口被占用
修改 `full_version.py` 最后一行：
```python
app.run(host='0.0.0.0', port=5001)  # 改为其他端口
```
