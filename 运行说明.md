# 快速运行指南

## 如果Python已正确安装

### 方法1：使用批处理文件（Windows）
1. 双击 `start_web.bat` - 启动完整版Web服务
2. 双击 `run.bat` - 运行命令行版本

### 方法2：手动启动
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动Web服务
cd web
python app.py

# 或运行测试版本（不需要API Key）
python test_app.py
```

## 如果Python环境有问题

### 解决方案1：使用Python在线版本
访问以下在线Python平台运行代码：
- https://replit.com
- https://www.python.org/shell/
- https://tio.run

### 解决方案2：安装Python
1. 访问 https://www.python.org/downloads/
2. 下载并安装Python 3.8或更高版本
3. 确保勾选"Add Python to PATH"

### 解决方案3：使用便携版Python
1. 下载WinPython：https://winpython.github.io/
2. 解压后使用内置的Python

## 测试Web界面

即使没有API Key，您也可以：
1. 运行测试版：`python web/test_app.py`
2. 访问 http://localhost:5000
3. 查看界面，测试所有功能（除生成图片外）

## 获取API Key

1. 访问 https://kie.ai/api-key
2. 注册账号并获取API Key
3. 创建 `.env` 文件并添加：
   ```
   KIE_AI_API_KEY=your_actual_api_key_here
   ```

## 故障排除

### 问题：找不到Python命令
- 尝试使用 `py` 命令代替 `python`
- 或使用完整路径：`C:\Python39\python.exe`

### 问题：pip不是内部命令
- 使用 `python -m pip` 代替 `pip`
- 或确保Python已添加到PATH

### 问题：端口被占用
- 修改 `app.py` 中的端口号（如改为5001）
- 或关闭占用端口的其他程序
